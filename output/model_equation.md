# Variable Identification and Data Preparation

## Dependent Variable

**monthly_sales**

The dependent variable represents the monthly sales generated by each retail store. This is the target variable that the regression models aim to predict.

---

## Independent Variables

The following variables were selected as predictors:

- marketing_spend
- footfall
- avg_discount_pct
- staff_count
- inventory_availability_pct
- competitor_distance_km
- holiday_flag
- customer_rating
- region
- store_type

These variables are expected to influence monthly sales either directly or indirectly.

---

## Numerical Variables

- marketing_spend
- footfall
- avg_discount_pct
- staff_count
- inventory_availability_pct
- competitor_distance_km
- customer_rating
- monthly_sales

---

## Categorical Variables

- region
- store_type

Categorical variables were converted into dummy variables before building the Multiple Linear Regression model.

---

## Data Cleaning Steps

The following preprocessing steps were performed before model development:

- Checked for missing values.
- Verified duplicate records.
- Confirmed numerical data types.
- Converted categorical variables into dummy variables.
- Verified target variable consistency.
- Checked for invalid or inconsistent values.

---

## Dummy Variables Created

### Region

- Region_East
- Region_West
- Region_North

South was treated as the reference category.

### Store Type

- Store_Type_Mall
- Store_Type_HighStreet

Standalone stores were treated as the reference category.

---

## Assumptions

- Monthly Sales is the dependent variable.
- Predictor variables are assumed to have a linear relationship with monthly sales.
- Dummy variable trap was avoided by dropping one category from each categorical variable.
- Regression assumptions were validated using residual analysis.
