# Business Analytics Capstone Project - Part 3

# Regression-Based Business Insights and Model Interpretation

---

## Project Overview

This project focuses on building and evaluating regression models to identify the key factors influencing monthly sales across retail stores. The analysis includes data preparation, feature selection, regression modeling, residual analysis, model comparison, and business recommendations.

---

## Business Problem

A retail company wants to understand which business factors have the greatest impact on monthly sales. By applying regression analysis, the company aims to identify the most influential variables and make data-driven decisions to improve overall business performance.

---

## Dataset Information

**Dataset Name:** business_regression_data.xlsx

The dataset contains operational and business performance information for multiple retail stores.

---

## Objective

The objectives of this project are:

- Clean and prepare the dataset.
- Identify independent and dependent variables.
- Build a Simple Linear Regression model.
- Build a Multiple Linear Regression model.
- Compare model performance.
- Perform residual analysis.
- Interpret regression coefficients.
- Provide business recommendations based on the findings.
